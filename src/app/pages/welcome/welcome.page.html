Powstał projekt końcowy dla studiów podyplomowych "Programista Front-end" na WSB Wrocław "Aplikacja internetowa z użyciem rest-api"....

Do stworzeniena aplikacji wykorzystałem Ionic Framework + Angular 7.

Jako REST API wykorzystałem swapi.co oraz themoviedb.org.

Aplikacja wykonołem jako PWA a więc można ją zainstalować na urządzeniach mobilnych oraz niektórych przeglądarkach desktopych.
